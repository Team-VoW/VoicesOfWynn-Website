# Voices of Wynn

Repository for the Voices of Wynn project. A Wynncraft mod that adds voiced NPC dialogue.

## Project Areas

- `website/` contains the PHP website and custom MVC framework. Follow the nested instructions there for website changes.
- `vow-api/` contains the .NET API. Follow the nested instructions there for API changes.
- `app-frontend/` contains the Vue 3 frontend application.
